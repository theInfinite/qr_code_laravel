

# QR Code

*By [sahil soni]*

Generate QR Code in Laravel Framework. Makes use of [simplesoftwareio/simple-qrcode](https://packagist.org/packages/simplesoftwareio/simple-qrcode)

## Installation

Use [Composer](https://getcomposer.org/) to install the library.

``` bash
$ composer update
```
